Tutorial. Designing and training a two-layer neural network

1 Creating a new project

Start Neural Network Console, click '+ New Project' and input name 'daemon' to create a new project.

2 Assigning the datasets

On the DATASET tab, there are no dataset for Training and Validation yet.

 

Click Training and 'Not Set' button, and assign mnist.tsmall_mnist_4or9_training to Training

 

Click Validation and 'Not Set' button, and assign mnist.small_mnist_4or9_test to Validation.

 

3 Designing the two-layer neural network

  1. Under “Component” on the left side of the EDIT tab, double-click on Input, Affine, Tanh, Affine, Sigmoid, and BinaryCrossEntropy, one by one, in order to add layers to the network graph.

Note

“Affine” is in the category Basic, “Tanh” and “Sigmoid” are in Activation, “BinaryCrossEntropy” can be found under Loss.

 

  1. Click Input layer to select it. Input “1,28,28” (the default value), which is the size of variable x (input data), in the Size box (click and input) under Layer Property.

 

  1. Click the first Affine layer in order to select it. Set the number of intermediate layer nodes (e.g., 100, the default value) in the OutShape box under Layer Property.

 

  1. Click the second Affine layer in order to select it. Set the number of output layer nodes (1, which is the size of variable y (output data) in this example) in the OutShape box under Layer Property.

 

4 Executing training

Click the Run Training button. training will start.
