Tutorial. Import a dataset to the Neural Network Console

1 Preparing a dataset folder

The dataset folder must contain a file(the file name must be index.csv) and a folder(the folder name must be data). For details, see “Dataset preparation”.

And we can get the contents of the dataset folder, as follows:

 

2 Import dataset

Start Neural Network Console, click 'Import Dataset' button, copy the destination path， and then copy or move the dataset folder to the destination path.

 

Input the fdataset folder name just now. and then click 'Start' button.

 

3 Previewing the new dataset

A preview of the selected dataset appears on the right side of the window.

And use this dataset in the project now.
